# 未完了タスク管理アラーム

Googleスプレッドシートの未完了タスクを監視し、期日の10日前、5日前、2日前にアラームメールを送信するGoogle Apps Script (GAS) ツールです。

## 機能
- **監視対象**: スプレッドシートの全行（1行目はヘッダー扱い）
- **期日**: E列の日付
- **通知先**: `reservation@wakamatuya.co.jp`
- **通知タイミング**:
    - **10日前**: 「再確認等は必要ありませんか？」
    - **5日前**: 「もう一度確認しましょう」
    - **2日前**: 「最終確認は出来てますか？」

## 導入手順

### 1. スクリプトの導入
1. 対象のスプレッドシートを開きます。
2. メニューの **[拡張機能]** > **[Apps Script]** をクリックします。
3. エディタが開いたら、`Code.gs` の内容をすべてコピー＆ペーストします。
4. プロジェクト名をつけて保存します。

### 2. トリガーの設定
毎日自動実行するためにトリガーを設定します。
1. 左メニューの **トリガー（時計アイコン）** をクリック。
2. **[+ トリガーを追加]** をクリック。
3. 以下の設定で保存します。
    - **実行する関数**: `checkDeadlinesAndNotify`
    - **イベントのソース**: `時間主導型`
    - **イベントのタイプ**: `日付ベースのタイマー`
    - **時刻**: `午前 9時 ～ 10時`（任意）
